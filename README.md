# Next.js fullstack authentication template

## Requisites
- Node.js v18+
- MongoDB instance
- Mailtrap account and keys

## Run on your machine
Clone the project

```bash
  git clone https://github.com/henrilab/next-auth
```

Go to the project directory

```bash
  cd nextauth
```

Use the correct version of Node

```bash
  nvm use
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
